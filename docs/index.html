GDA Toolbox (public version)¶

This package provides easy-to-use tools for manipulating time-series signals (see the timeseries.Signal class), point clouds (see the multidim.PointCloud class) and simplicial complexes (see the multidim.SimplicialComplex class). The goal is to provide intuitive interfaces and fast, practical, and mathematically-correct methods for topological data analysis. The basic capabilities are:

  • 0-Dimensional Persistence via distance.
  • 1-Dimensional Persistence via the Rips or Cech complex.
  • CoverTree with Friends for density estimation.
  • CDER for supervised learning of labelled, weighted pointclouds

Modules¶

multidim User-facing tools for topological data analysis.
multidim.models
timeseries This module defines the Signal class, the basic object for measuring time-series signals using zero-dimensional persistence of graphs.
homology This module defines the PersDiag class, the basic object for persistence diagrams.

Fast Algorithms¶

Performance-sensitive algorithms are written in Cython. Usually, only core developers need to access these methods directly, as most users will use the pure Python interfaces for convenience.

multidim.fast_algorithms Fast routines for multidimensional topological data analysis.
timeseries.fast_algorithms This stanza is for pytest discovery
timeseries.curve_geometry Fast routines for space curves and time-series data.
